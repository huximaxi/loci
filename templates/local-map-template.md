## How to read this map

The palace is a **two-tier memory system**:

- **Local palace** (`rooms/` etc.) = dev-staging working memory. Where your AI thinks.
- **Knowledge core** (`[palace-name]/`) = soul files, skills, handovers, garden.
- **Global layer** (`~/.claude/CLAUDE.md`) = behavioral constants. Never holds state.

The architectural rule: **if it changes session to session, it belongs in Layer 1. If it describes how to behave, it belongs in Layer 0.** Mixing the two causes drift.

The handover chain is the critical reading path for session continuity. If context is lost, re-read layers 0→2 in order.
