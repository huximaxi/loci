# loci tutorials

Short, do-it-now walkthroughs. Each one uses templates that already ship in `templates/`.
No build required: copy the template into your palace and point any file-aware AI at it.

Pick by what you want to do, or run the feature helper (`templates/feature-helper.md`) and
let it point you.

| Flow | Feature set | You end with |
|---|---|---|
| [your-first-crystal](your-first-crystal.md) | Persistent Memory | a tiered fact your AI treats as ground truth |
| [build-a-room](build-a-room.md) | Context Architecture | a room that loads the right context on open |
| [name-your-companion](name-your-companion.md) | Identity & Personas | a soul file and a peer card |
| [grow-the-garden](grow-the-garden.md) | The Garden | a planted idea and a health pass |
| [close-and-reopen](close-and-reopen.md) | Continuity & Synthesis | a handover that reopens warm |
| [set-a-gate](set-a-gate.md) | Trust & Governance | a review gate and a quarantine rule |
| [go-cross-tool](go-cross-tool.md) | Interop & Evolution | memory that travels across tools |

One flow per feature set. Start anywhere; each one ends with something working in your palace.
