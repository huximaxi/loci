# discoveries

A community folder. Each entry is a palace portrait — a curated snapshot of how someone built and uses their loci palace.

These are not bug reports or feature requests. They are records of what the framework became in someone else's hands.

---

## What belongs here

- Unexpected use cases (fiction, research, teams, creative work)
- Modifications to the baseline methodology that turned out load-bearing
- Crystal tier adaptations, gate configurations, new output primitives
- Honest notes on what's missing

What does not belong here: sensitive content, personal data, anything you would not post publicly.

---

## How to submit

**Via pull request:** copy `TEMPLATE.md`, fill it in, save it as `discoveries/[your-handle].md`, and open a PR. That's it.

**Via email:** send a filled-in template to themapisnory@tuta.io. We'll add it with your permission.

Include only what you're comfortable making public and permanently visible. Once merged, entries live in git history.

---

## What we do with these

Read them. They feed directly into what gets built next. With explicit permission, selected portraits may appear on loci.garden as use case examples.
